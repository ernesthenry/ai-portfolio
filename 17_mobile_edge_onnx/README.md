# Mobile & Edge Deployment (ONNX)

**Goal:** Deploy AI models to "Apps" (iOS, Android, Web) without needing a Python server.

**The Solution:** ONNX (Open Neural Network Exchange).

1.  **Train** in PyTorch/Python.
2.  **Export** to `.onnx`.
3.  **Run** on the edge using `ONNX Runtime` (C++, C#, Swift, JS).

**Why this matters:**
Real AI Engineers know that 90% of apps cannot run a Python backend on the user's phone. ONNX is the bridge.
