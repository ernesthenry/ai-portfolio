# PyTorch Fundamentals

**Goal:** Prove understanding of Core ML math (Gradient Descent, Backprop) without high-level wrappers.

## Modules

1. **Linear Regression (`linear_regression.py`)**:

   - Predicts house prices.
   - Uses `MSELoss` and `SGD`.
   - Demonstrates understanding of continuous value prediction.

2. **Logistic Regression (`logistic_regression.py`)**:
   - Classifies emails as Spam/Not Spam.
   - Uses `BCELoss` (Binary Cross Entropy) and `Sigmoid` activation.
   - Demonstrates understanding of binary classification.
