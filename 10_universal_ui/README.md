# Universal UI Wrappers

These two apps provide a "Frontend" for your backend engineering portfolio.

## 1. Streamlit (`app_rag.py`)

Best for RAG and specialized tools (Summary, Routing).
Run: `streamlit run app_rag.py`

## 2. Chainlit (`app_chat.py`)

Best for Chat Agents and Streaming interaction.
Run: `chainlit run app_chat.py -w`
